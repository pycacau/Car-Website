ApexDrive Collection - Car Website
==================================

Projeto de landing page moderna para uma coleção de supercarros de luxo, com carrossel animado, design responsivo e foco em visual premium.

## Tecnologias

- HTML5
- CSS3 (layout responsivo, animações e gradientes)
- JavaScript (carrossel com autoplay e navegação por setas/indicadores)

## Funcionalidades

- Carrossel em tela cheia com **5 modelos de carros**.
- Animações suaves de transição entre os slides.
- Botões de navegação (anterior/próximo) e indicadores clicáveis.
- Seção de contato com atalhos para **e-mail** e **WhatsApp**.

## Como executar o projeto

1. Faça o clone ou download deste repositório.
2. Abra a pasta `Car-Website` no seu editor.
3. Abra o arquivo `index.html` diretamente no navegador  
   (duplo clique ou botão direito → "Abrir com" → seu navegador).

Não há dependências externas ou build: é um projeto 100% estático.

## Estrutura dos arquivos

- `index.html` – estrutura da página e conteúdo dos carros.
- `style.css` – estilos, layout, responsividade e animações.
- `script.js` – lógica do carrossel (troca de slides, autoplay e cliques).
- `img/` – imagens dos carros, setas de navegação e logotipo.

## Personalização rápida

- Para trocar os carros, substitua as imagens na pasta `img/` (`1.png` a `5.png`).
- Edite os textos de cada modelo em `index.html` (títulos, descrições e tagline).
- Ajuste cores, fontes e intensidades de efeitos em `style.css`.
